# Проект: Мой магазин на Django

## Суть приложения
Это учебное приложение для изучения основ работы с Django: создание проекта, настройка маршрутов и отображение простых страниц.

## Список страниц
1. Главная (/) — ссылки на остальные разделы.
2. Об авторе (/author/) — информация о создателе проекта.
3. О магазине (/store/) — информация о теме лабораторной работы и таблица.
